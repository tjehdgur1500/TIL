#### 4.Tuple

```python
tuple()
t = ()
t = 1,
t = 1,2,
```

튜플은 리스트와 매우 비슷한 형태를 띄고 있다. 그저 괄호만 다를 것 같다고 생각할 수 있다.
왜냐하면 값이 똑같이 들어가고 순서를 유지하고 있지만 조금 다른점이 있다.
큰 특징으로는 수정 X , 삭제 X , 순서 O

즉 튜플은 리스트 처럼 수정하거나 삭제를 할 수 없는 형태이다.
사용하는 예를 들자면 도시의 좌표 같은게 될 수 있겠다. 좌표는 바뀌면 안되니 수정 , 삭제가 되지 않게 만드는것이다.

```python
# 임의로 만든 좌표
location = (514112, 618515161)
```

또 다른 점이 있는데 튜플은 () 괄호를 꼭 쓰지 않아도 생성이 가능하다.
어떤 값 앞에 , 만 찍어주면 튜플로 자동 캐스팅 된다.

```python

tu = "a",
```
