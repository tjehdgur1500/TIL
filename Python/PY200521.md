- 클래스 

### 1. 클래스

```python

class hero :
    def __init__(self, name, power) :
        self.name = name
        self.power = power

    def heroInfo(self):
        print(f"{self.name} | {self.power}")

c1 = hero("a", "1000")
c1.heroInfo()
```

파이썬에서는 클래스를 생성할때 위와 같이 만든다.
클래스는 재사용성이 굉장히 높다.

하나의 클래스를 호출해서 변수에 할당하면 그 변수는 해당 클래스에 접근이 가능하다.

이런경우를 인스턴스화 한다고 한다.
인스턴스화 된 변수는 클래스 자기자신을 칭하기 때문에
호출부분에 인자를넣어 매개변수들에 접근하여 사용 할 수 있고
함수도 접근해서 사용할 수 있다.

* 같은 클래스로 여러 인스턴스를 생성할 수 있다.

```python
c2 = hero("b", "500")
c3 = hero("c", "1500")
```
이렇게 접근하면 공통 클래스를 가진 인스턴스들을 사용하여
코드 재사용하는데 용이해진다.

### 2. 부모 클래스 호출

부모 클래스 호출하는 방법이다.
클래스는 자식클래스를 생성 할 수도 있다.
자식 클래스를 생성 할 수 있다는 것은 부모 클래스를 상속 받을 수도 있다는 말이다.

```python
class Aclass :
    def __init__(self, ob) :
        self.ob = ob

class Bclass(Aclass) :
    def __init__(self,ob, li) :
        super().__init__(ob)
        self.li = li

a = Aclass("func")
b = Bclass("func", "list")

a.ob >> "func"
b.ob >> "func"
b.li >> "list"

```

위 코드를 보면 Aclass , Bclass 가 존재한다.
이 둘의 차이점은 Bclass를 보면 알 수 있다.
Bclass 에서 매개변수로 Aclass를 상속 받고 있다.
그렇다는 것은 Bclass 는 Aclass의 객체들을 모두 사용할 수 있다는 말이다.

또한 Bclass는 Bclass 자체에서 또다른 매개변수를 생성해서 Bclass를 확장 시킬 수도 있다.

주의할 점은 부모클래스인 Aclass 에서는 자식 클래스의 새로운 객체에 접근하지 못할것이다. 상속을 준것이지 자식클래스를 사용할 수 없는것이다.
