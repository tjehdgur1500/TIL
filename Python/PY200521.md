- 클래스 

### 1. 클래스

```python

class hero() :
    __init__(self, name, power) :
    self.name = name
    self.power = power

    def heroInfo(self):
        print(f"{self.name} | {self.power}")

c1 = hero("a", "1000")
c1.heroInfo()
```

파이썬에서는 클래스를 생성할때 위와 같이 만든다.
클래스는 재사용성이 굉장히 높다.

하나의 클래스를 호출해서 변수에 할당하면 그 변수는 해당 클래스에 접근이 가능하다.

이런경우를 인스턴스화 한다고 한다.
인스턴스화 된 변수는 클래스 자기자신을 칭하기 때문에
호출부분에 인자를넣어 매개변수들에 접근하여 사용 할 수 있고
함수도 접근해서 사용할 수 있다.

* 같은 클래스로 여러 인스턴스를 생성할 수 있다.

```python
c2 = hero("b", "500")
c3 = hero("c", "1500")
```
이렇게 접근하면 공통 클래스를 가진 인스턴스들을 사용하여
코드 재사용하는데 용이해진다.
