- 반복문

### 3.반복문

```python

while
for in

```

반복문은 두가지로 나뉜다.

1. while
2. for in

```python
i = 0
while i < 10 :
print(i)
i += 1
```

while 문은 조건을 기준으로 반복 실행한다.
개인적으로 while 쓰는게 조금 불편하지만 조건하에 반복할때 유용하게 쓰인다.

while은 주의할점이 있다 . 바로 무한루프라는건데 이코드를 작성시
항상 True 이기 때문에 조건에서 벗어나지 못해 반복을 무한히 하게된다.
그럼 결국 컴퓨터가 과부하(?) 하여 치명적인 상황이 벌어진다.

```python
while True :
    print("loop !")
```

```python
for i in range(5) :
    print(i)
```

for문은 범위를 기준으로 반복한다.
범위는 range 메서드로 정해줄 수 있는데 고정값 정수가 아닌 유동적인 값을 넣을 수 도 있다.
ex) int(input())
