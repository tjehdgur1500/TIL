#### 1.Boolean

boolean 은 모든 언어에서 존재하며 True , False 값으로 존재한다.
말 그대로 참 , 거짓 값이며 이진법으로 바꾸면 아래와 같다.
0 = False
1 = True

```python
if 1 == True :
    print(True)
elif 0 == True :
    print(True)
else :
    print('0 is False')
```

답을 예상해보면 1은 True이기 때문에 True가 출력될것이고
0은 False이기 때문에 else 문의 '0 is False'가 출력될것이다.
간단하게 알아보았지만 0 과 1 로만 참 , 거짓이 허용되는것은 아니다.
아마 앞으로 정리하면서 자연스레 나올것 같다.

#### 2.String

String 이라 하면 문자열을 뜻한다.
문자열은 보통 'Single Quotes' , "Double Quotes" 로 감쌀 수 있다.
문자열 또한 값이기때문에 참이다.

```python
print('Hello World')
print("Hello World")
print("""
    String Here !
""")
print("Hi 'A'")
```

세번째 예제 같은 경우는 """ 따옴표를 세개 붙여서 쓴다.
파이썬에서는 print 문 안에서 줄바꿈이 허용되지 않는다.
허용되는 경우는 세개의 따옴표안에서는 허용이 되기 때문에 개행해도 출력에 문제가 없다.

4번째 예제는 작은따옴표 혹은 큰따옴표를 문자열안에서 출력하고 싶을때 사용하는 방법이다.
문자열은 중복되서 따옴표를 넣을 수 없기 때문에 중복되지 않는 따옴표를 사용하면 그 따옴표를 문자열로 인식한다.
