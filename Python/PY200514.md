### print()

```python
a = "hello python"
print("hello Python")
print(a)
```

print 함수는 말그대로 화면에 출력해주는 함수이다.
간단하지만 유용하게 많이 사용된다.

위의 예시처럼 string 값 그대로 넣어 출력할 수도 있고
변수를 넣어 변수에 할당된 값을 출력 시켜줄 수도 있다.

사실 이렇게 쓸 수도 있지만 더 많은 상황에서 쓰인다.
아래 format 같은 경우이다.

### format

```python
a = 'Seo'
b = 25
print("my name is %s age is %d" %(a, b))
# my name is Seo age is 25
```

이렇게 문자열 포맷을 통해 % 표현식으로 값을 넣어 변수를 할당해 줄 수 있다.
%s = 문자열 , %d = 정수 , %f = 소수점

이렇게 표현이 가능하다.
이 방법은 제일 오래된 방법이다. 하지만 알고 있으면 좋음 !

이 다음 업그레이드 된 방법으로는 format()이 있다.

```python
a = 'Seo'
b = 25
print("my name is {} age is {}".format(a,b))
# my name is Seo age is 25
```

명확하게 어떤 값이 들어갈지 보여주고 있다.
마지막으로 f.string 이다.

```python
a = 'Seo'
b = 25
print(f"my name is {a} age is {b}")
# my name is Seo age is 25
```

세번째 방식은 파이썬 3.6 버전 이상에서 작동하는 비교적 최신 문법이다.
더욱 간편해졌고 직관적으로 표현이 가능한 장점이 있다.
