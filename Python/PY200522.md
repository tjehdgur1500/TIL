### 1. 클래스 메서드

- 클래스 와 인스턴스의 차이

1. 클래스가 가지고 있는 특징

클래스 메서드와 클래스 변수를 가진다.

```python

class Info :
    variable = 'python var'

def __init__(self,lang) :
    self.lang = lang

@classmethod
def clsMethod(cls. logs) :
    print(logs)

var1 = Info("python")
print(var1.lang)
print(var1.variable)
Info.clsMethod("is Workd!")
# >> "python"
# >> "python var"
# >> "is Worked!"

```

클래스에서 init 이 전에 작성한 변수가 바로 클래스 변수이다.

클래스 변수의 특징은

클래스에서 접근이 가능하면서 자식클래스 혹은 다른 객체에서도 접근이 가능하다.

일종의 전역변수 같은건데 클래스 내부에서만 허락한다.

클래스 메서드는 인자를 self 가 아닌 cls 를 가지며 이름은 관용적으로 쓰는것이다.

이 cls 는 class를 받고 있어 클래스 메서드 자체는 클래스의 메서드 인것이다.

접근할때는 위처럼 클래스.클래스메서드로 접근하는게 논리적으로 맞다.

2. 인스턴스가 가지는 특징

```python
class Info :
    variable = 'python var'

def __init__(self,lang) :
    self.lang = lang

def move(self) :
    print(self.lang, "go")


var2 = Info("javascirpt")
var2.move()
var2.lang
```

인스턴스를 정리하기위해 간단하게 클래스를 작성했다.

var.move() 는 인스턴스 메서드고

var.lang 은 인스턴스 변수이다.

즉 class에서 생성하고 self를 인자로 받는 객체들은 모두 인스턴스다.
