 ## linux commend

 ### 꼭 알고가면 좋을 리눅스 커맨드

 우선 CLI , GUI 가 존재한다.<br>
 컴퓨터의 명령처리를 명렁어로 하는 CLI <br>
 우리가 보는 화면에서 마우스로 클릭해 몀령을 수행하는 GUI<br><br>

 물론 터미널에서 작업하는건 매우 불편할 수도 있으나 , <Br>
 적응이 되면 마우스를 쓰는게 얼마나 불편하고 피곤한 일이였는지 깨닫게 된다.<br><br>
 자주 쓰이는 커맨드를 몇개 기록해보려한다. <br>

#### cd

 ```
 cd dirname
 ```
 디렉토리를 이동하는 명령어다 . <br>

#### ls
 ```
 ls
 ```
 현재 위치한 곳에서 파일 리스트를 나열한다. <br>

#### rm
 ```
 rm -rf
 ```
 삭제 명령어다 . <br>
 뒤에 option을 붙여 파일삭제 , 폴더삭제를 지정해서 사용할 수 있다. 

#### pwd
 ```
 pwd

 /Users/mellomasi

 ```

 현재경로가 어디에 위치해있는지 알려준다. <br>
 
 #### ~ , /
 ```
~
/
 ```
 - ~ : 사용자 home 디렉토리로 이동한다.
 - / : 최상단 루트로 이동한다.

#### cp
```
cp test test1

ls
dir/ test test1
```
복사할때 사용하는 명령어다.<br>
cp -r 을 붙여 디렉토리를 복사할 수 있다.

#### mkdir
```
mkdir newDir

mkdir a/b/c/d/

```
새로운 디렉토리를 생성한다.<br>
-p 옵션을 붙여 하위 디렉토리도 생성한다.

#### touch
```
touch file.js
ls
dir/ file.js
```
새 파일을 생성한다 . <br>
뒤에 dot 로 확장자를 정하여 생성할 수 있다.

#### cat
```
cat test

ls
1

cat test > test2
ls
test test2
```

cat으로 해당파일을 출력할 수 있고 , <br>
해당 파일을 복사할 수도 있다.<br>
여러모로 유용하게 사용될듯 하다.
