- 생성한 Templates을 화면에 렌더링 하기 위해 프로젝트의 urls에 url을 추가하는 방법을 알았다.

- 해당 앱에서도 urls 를  생성해서  
  url 경로를 만들어 첫번째 인자로 넣어주고
  views 에서 작성한 템플릿 생성을 요청하는
  render를 반환하는 views에서 가져온 함수를 두번째 인자로 넣어준다.

```{.python}

from django.urls import path
from . import views

# views에서 가져오고 있음

urlpatterns = [
  path('register/', views.register),
]

```

```{.python}

# views 에서 템플릿 비즈니스로직 처리를 위해 만든 함수
# 기본적으로 request를 인자로 받고 있다.
# django의 shortcuts 모듈이 render 기능을 사용하는것

from django.shortcuts import render

def register(request) :
  return render(request, 'register.html')

```
