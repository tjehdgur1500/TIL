### **문제 설명**

길이가 n이고, "수박수박수박수...."와 같은 패턴을 유지하는 문자열을 리턴하는 함수, solution을 완성하세요. 예를들어 n이 4이면 "수박수박"을 리턴하고 3이라면 "수박수"를 리턴하면 됩니다.

### 제한 조건

- n은 길이 10,000이하인 자연수입니다.

### 입출력 예

|n|return|
|:---:|:---:|
|3|"수박수"
|4|"수박수박"

```jsx
function solution(n) {
  let word="";
  for (let i=1; i <= n; i++){
      if (i % 2 === 1){
          word = word + '수'
      }else{
          word = word + '박'
      }   
  } 
    return word
}
```

문자열의 길이에 따라 달라지는 규칙이 있는데 , 

문자열이 홀수이면 수 , 짝수이면 박을 리턴하면 된다.

하지만 n 만큼 반복해서 출력해야하므로 빈문자열 변수를 만들어 n이랑 i가 같아질때까지 문자열을 연결해서 리턴한다.
