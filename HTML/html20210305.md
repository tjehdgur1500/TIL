## HTML (Hyper Text Markup Language)

HTML은 예전부터 오늘날까지 웹브라우저에서 사용되는 문서이다.<br><br>

HTML 페이지의 구조를 만드는것이고 배우기가 그다지 어렵지 않다. <br>
어렵지 않다고 진짜 어렵지 않은것은 아닌것 같다고 생각했다. <br><br>

> HTML은 wJavascript처럼 반복문이나 조건문을 쓸 수 없다.  <br>
> 물론 Js 의 라이브러리 , 프레임워크들의 템플릿 엔진을 쓰면 사용가능하지만 이건 엄연히 html은 아니니.. ( ejs, pug) 등 <br>

<br><br>

### html의 구조 
<br>

```html
<!DOCTYPE html>
<html>
<head>
<title></title>
</head>
 <body>
    <div>
        <h1>Hello HTML</h1>
        <p>HTML5 is simple</p>
    </div>    
 </body>
</html>
```
우선 간단한 구조의 HTML 문서이다. <br>
위에서부터 차례차례 읽어 나가보려한다. 

```html
<!DOCTYPE html>
```
DOCTYPE 이란 html 문서임을 알리는 시작점이자 html 버전을 나타낸다 . <br>
최신 버전 html5 는 위와 같이 !DOCTYPE html 으로 선언을 한다.  <br>
물론 이전 버전도 있다. <br><br>

HTML 4.01 Strict

```html
<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd">
```


XHTML 1.0 Strict
```html
<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
```

예전 버전들의 doctype 선언태그이다. <br>
하지만 위의 것들은 그냥 이런것들이 있다 정도만 아는게 좋을것인게 이 버전들은 이제 쓰이지 않는다. <br><br>

```html

<html> <!-- html의 시작을 알리는 태그 -->
    <head> <!--html의 각종 정보를 담는 태그-->
      <title></title> <!--브라우저에서 보여줄 문서명-->
    </head>
    <body> <!-- 실제 화면에 보일 코드가 시작하는 태그-->
    </body>
</html>
```

잘 살펴보면 특징이 있다. <br>
바로 열고 닫는 <열고> <닫는태그/> 이다. <br>
이 태그들은 꼭 작성이 끝나면 닫아주도록 한다. <br><br>


### Tag

html에는 수많은 태그가 존재하지만 현재 사용하는 태그는 그렇게 많지 않다. <br>
어떤 태그가 대표적으로 많이 사용되는지 알아보자면 <br><br>

```html
<div>
    한줄을 차지하는 division 

    <h1>문서의 타이틀이 되는 태그</h1>
    <p>문단 태그</p>
    <span>자기자신 외에 다른 태그가 옆에 올 수 있다.</span>
    <ul>
        리스트의 그룹 ( 부모격 )
        <li>리스트</li>
        <li>리스트</li>
    </ul>
    <img src="">
</div>
```
각 태그들을 보면 언제 사용해야하는지를 알 수 있다. <br>

### Semantic Tag

위의 태그이외에 시멘틱 태그라 불리는 태그들도 존재한다. <br>

```html
<article>
    <section>
      <h1>안녕하세요</h1>
    </section>
</article>
```

article , section 같은 태그가 시멘틱태그다 . <br>
딱히 무슨 기능이 있진 않지만 의미를 부여 할 수있다는 게 차이점이다. <br><br>

html은 구조를 표현하기 위한 언어다. <br>
이런 시멘틱 태그를 이용하면 구조를 명확히 할 수 있으며 무엇을 의미하는지 구별을 할 수 있다.

- div 와 section 차이

   > 대표적으로 비교할만한 태그는 div 와 section이다. <br> <br>
   > - div 는 영역분할을 위한 태그이다. <br>
   > - section 은 독립적인 구획을 나타내는 태그이다. <br><br>


별 차이 없어보이지만 시멘틱 태그들은 각자의 역할이 있어 <br>
사용하는것이 좀더 좋은 문서를 만들 수 있다. <br>
물론 div를 사용하면 안되는것은 아님<Br><BR>
